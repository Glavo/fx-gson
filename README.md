Gson extension to make it easy to deal with JavaFX properties.

## Usage

### Add the dependency in Gradle

    repositories {
        maven {
            url  "http://dl.bintray.com/joffrey-bion/maven"
        }
    }

    dependencies {
        compile 'org.hildan.fxgson:fx-gson:1.0'
    }
